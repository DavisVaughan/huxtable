Huxtable is a package for creating HTML and LaTeX tables. It provides similar functionality to xtable, but more, with a simpler interface. Here's a quick example. 

ht <- hux(
        Employee = c('John Smith', 'Jane Doe', 'David Hugh-Jones'),
        Salary = c(50000, 50000, 40000),
        add_colnames = TRUE
      )

bold(ht)[1,]           <- TRUE
bottom_border(ht)[1,]  <- TRUE
align(ht)[,2]          <- 'right'
right_padding(ht)      <- 10
left_padding(ht)       <- 10
width(ht)              <- 0.35

ht
Employee Salary
John Smith 50000.00
Jane Doe 50000.00
David Hugh-Jones 40000.00

To learn more, check out the vignette in HTML or PDF format, or the original R Markdown.

Installation

At the moment, huxtable is only available from github:

library(devtools)
install_github('hughjonesd/huxtable')

The logo

Here's the code to generate the logo at the top of the page.

library(huxtable)
logo <- hux(c('h', NA), c('u', 'table'), c('x', NA))
rowspan(logo)[1, 1] <- 2
colspan(logo)[2, 2] <- 2
logo <- set_all_borders(logo,,,1)
font_size(logo) <- 20
font_size(logo)[1, 2:3] <- 24
font_size(logo)[1, 1] <- 54
background_color(logo)[1, 1] <- '#e83abc'
background_color(logo)[1, 3] <- 'black'
text_color(logo)[1, 3] <- 'white'
width(logo) <- '140pt'
font(logo) <- 'Palatino,Palatino Linotype,"Palatino LT STD",Book Antiqua,Georgia,serif'
top_padding(logo)[1, 1] <- 5
bottom_padding(logo)[1, 1] <- 5
left_padding(logo)[1, 1] <- 10
